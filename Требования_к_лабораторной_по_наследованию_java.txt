Требования к лабораторной работе по наследованию (java)

Создать консольное приложение, удовлетворяющее следующим 
требованиям:
1) Использовать возможности ООП: классы, наследование, полиморфизм, 
инкапсуляция.
2) Каждый класс должен иметь отражающее смысл название и 
информативный состав (поля, set, get, конструкторы, toString).
3) Наследование должно применяться только тогда, когда это имеет 
смысл.

 - один интерфейс 
 - абстрактный класс 
 - один внутренний класс
 - перечисление
 - сбора объектов использовать стандартные коллекции
 - консольное меню
